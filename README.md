# Cmd_Powershell.md
## Les commandes PowerShell qui correspondent aux commandes Linux (shell bash)

_**cp**_ : Copy-Item

_**rm**_ : Remove-Item

_**cd**_ : Set-Location

_**mkdir**_ : New-Item

_**man**_ : Help

_**history**_ : Get-History

_**alias**_ : Set-Alias

_**cat**_ : Get-Content
